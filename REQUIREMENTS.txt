* CUDA-enabled GPU acceleration

* CellViT++ cellular representation learning
* Multimodal deep learning pipelines
* XGBoost classification and survival modelling
* Multi-GPU distributed training
* Whole-slide image (WSI) processing
* Large-scale computational pathology workflows

🧠 Explainability and Model Interpretation

The framework integrates explainability methods for both image-based and structured data analysis.

SHAP Analysis

SHAP (SHapley Additive Explanations) was used to evaluate the contribution of:

* Clinicopathological variables
* Nuclear morphometric descriptors
* Multimodal features
* Survival predictors

The explainability pipeline supports:

* Global feature importance
* Local prediction explanations
* Survival model interpretation
* Feature interaction analysis

Attention-Based Interpretability

Attention mechanisms implemented within:

* UNI
* Virchow
* CellViT++
* Multiple Instance Learning (MIL)

were used to identify diagnostically relevant histopathological regions.

The interpretability pipeline supports:

* Attention heatmaps
* Feature attribution analysis
* Morphological pattern exploration
* Spatial tumour visualisation

🧬 Foundation Models

The framework includes support for:

* UNI (Mahmood Lab)
* Virchow (Paige AI)
* CellViT++

These models were used as feature extractors and cellular representation learners without modification of their original architectures.
